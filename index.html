Why I Choose Uplift

My journey to learn coding began when I read Satoshi Nakomoto's White Paper on Bitcoin. It was then that I realized how the language of coding can be used to impact the world in a beautiful way. I wanted to learn to read and write code ever since. After quite a bit of self-study, I found that I was bombarded with information overload and did not know where to begin. In comes Uplift, and the application process alone motivated me to create this page from scratch, and immediately I am uplifted.
